rust_spreadsheet

The real spreadsheet engine (formula evaluation, dependency tracking, undo/redo, viewport scrolling) compiled to WebAssembly and running entirely in this browser tab -- no server involved.

Click a cell, type a value or formula into the top bar, press Enter. Formulas: A1+B2, A1-B2, A1*B2, A1/B2, SUM(A1:A10), AVG(...), MIN(...), MAX(...), STDEV(...). Referencing another cell auto-tracks the dependency, so changing A1 recalculates anything that depends on it. Circular references are detected and rejected.

The grid you see is a window into a much larger sheet (100x100 by default, resizable with "New sheet" above) -- the Command box sends raw commands straight to the engine (same syntax as the original CLI), including W/A/S/D to scroll that window and SCROLL_TO <cell> to jump to it directly.
